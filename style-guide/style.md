## 속성 줄여쓰기 - Shorthand Properties

→ 속성을 축약하여 사용한다. (except: font)

## 단위, 숫자

→ 0값은 단위를 붙이지 않는다.

→ 소수점일 경우 앞에 0.을 붙인다.

→ 색상일경우 3자리 가능하면 표기

→ border: none →0
